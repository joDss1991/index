Comment utiliser l'outil
1
Sélectionner vos appellations
Onglet Appellations : cochez les appellations utiles pour ce dossier. Elles apparaîtront dans les menus déroulants des parcelles.
2
Remplir l'en-tête dossier
Onglet Saisie, section Informations globales : contrat, assuré, commune, type de sinistre, expert…
3
Ajouter les parcelles expertisées
Section Comptages parcelles : bouton "+ Ajouter une parcelle" autant de fois que nécessaire. Un bloc par parcelle visitée.
4
Consulter la synthèse
Onglet Synthèse : détail par parcelle, total dossier, synthèse par appellation (surface cumulée, rendement moyen pondéré, production).
5
Exporter / Sauvegarder
Bouton Exporter PDF en bas pour le rapport. Excel pour remontée dans les systèmes. JSON pour sauvegarde et reprise.
💾 Comprendre les 3 formats d'export
Les données saisies sont enregistrées automatiquement dans votre navigateur. Pour les partager ou les mettre à l'abri, utilisez les formats ci-dessous via le bouton 📤 en haut à gauche.
📄
PDF — Rapport
Document figé prêt à imprimer, archiver ou transmettre à l'assuré / à la compagnie. Non modifiable.
📊
Excel — Remontée données
3 feuilles (Dossier, Parcelles, Synthèse). Utile pour remontée dans les systèmes Abeille ou analyses personnelles.
💾
JSON — Sauvegarde de travail
Seul format qui permet de reprendre un dossier là où vous l'avez laissé, sur un autre appareil ou plus tard. Indispensable avant "Nouvelle saisie" si vous voulez pouvoir revenir au dossier.
Cas d'usage typiques :
• Sauvegarder avant de changer de dossier (sécurité)
• Commencer sur tablette terrain, finir sur PC au bureau
• Transférer un dossier à un collègue expert
Bon réflexe : s'envoyer chaque JSON par mail et se créer un dossier dédié "Sauvegardes estimation viticole". En cas de problème (perte du téléphone, navigateur effacé), vous récupérez tous vos dossiers en quelques secondes.
Nommage automatique des fichiers
Les fichiers exportés sont nommés automatiquement selon le format :
AssuréNom-NuméroSinistre_DateHeure.extension

Exemple : Domaine-Exemple-S2026-1234_20260815_1430.pdf

Si l'Assuré ou le N° sinistre n'est pas renseigné, le nommage utilise ce qui est disponible (contrat, date).
